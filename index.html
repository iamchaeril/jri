Case Studies

Optimasi Jaringan Kantor Kontraktor HVAC di Bintaro Permai, Jakarta Selatan


Case:Klien mengalami WiFi sering putus dan aplikasi internal lambat.
Solusi:Kami melakukan survey lokasi, penataan ulang access point, load balancing Mikrotik dan penataan kabel LAN.
Hasil: Stabilitas meningkat 90% dan tidak ada downtime selama jam operasional.

Setup Kantor Baru di Cengkareng, Jakarta Barat


Case:Perusahaan 3 Lantai membutuhkan jasa setup IT lengkap seperti mesin absen fingerprint, 13 Titik CCTV, PABX & 12 Line Telpon, Jaringan Wifi, Server dan Printer Sharing.
Solusi: Kami lakukan instalasi LAN perlantai, Setup Acces Point Unifi perlantai, Konfigurasi Mikrotik untuk management jaringan, Instal Server Lokal untuk Aplikasi, Instalasi CCTV ditiap lantai, Instalasi Line Telepon dengan PABX Panasonic, Instalasi mesin fingerprint untuk absen karyawan dan Setup printer sharing, Windows dan Office
Hasil: Kantor siap digunakan dalam 3 hari, seluruh perangkat terhubung stabil.

Implementasi NAS Berbasis WD My Cloud untuk Kantor Building Management di Sudirman, Jakarta Selatan


Case:perusahaan mengalami permasalahan mengenai data yang tersimpan tidak terpusat, pernah terjadi data penting hilang karna system atau harddisknya rusak.
Solusi:Kami memberikan solusi untuk menggunakan Sistem Penyimpanan NAS yang aman dilengkapi RAID 1 dengan WD My Cloud
Hasil: User dapat mengakses data dimanapun dengan aplikasi bawaannya, tidak ada lagi file duplikat, bisa berkolaborasi dengan tim lain dan menurunkan risiko kehilangan data hampir 0% .

Preventive Maintenance Laptop dan PC Perusahaan Event Organizer di Ciputat, Tangerang Selatan


Case:Laptop dan PC untuk operasional mengalami kendala lambat, rendering lama dan sering Hang.
Solusi:Kami melakukan Diagnosa Performa & Health Check, Upgrade Komponen Penting, Optimasi System Operasi, Pembersihan Virus dan Malware, Cleaning Laptop dan PC dari Debu dan Ganti Pasta Processor, Reinstall OS(jika diperlukan) dan Memberikan Edukasi Pencegahan
Hasil: Laptop & PC Kecepatan meningkat lebih cepat, zero malware setelah pembersihan, Startup Windows kurang dari 12 detik, tidak ada lagi hang/not responding saat bekerja, produktifitas karyawan meningkat, menghemat biaya karena tidak perlu membeli komputer baru.

Setup Server Baru untuk Kantor Jasa Angkutan Logistic di Kapuk Kamal, Jakarta Utara


Case:Perusahaan membutuhkan akses data terpusat dan deploy aplikasi kantornya
Solusi:Kami lakukan instalasi dan konfigurasi server dengan Windows Server, setup database Microsoft SQL Server, deploy aplikasi, koneksikan Server ke jaringan LAN, Setup vpn remot, hingga implementasi sistem backup terjadwal.
Hasil: Data dan Aplikasi dapat diakses oleh Karyawan atau User kapanpun dan dimanapun dengan bantuan vpn.

Perbaikan dan Pencegahan Server Terinfeksi Ransomeware dikantor Kalideres, Jakarta Barat


Case:Server Perusahaan sering terkena ransomeware, sudah 2 install ulang sistem operasinya namun tetap terkena serangan ransomeware
Solusi:Kami melakukan audit keamanan server menyeluruh, menerapkan standar keamanan profesional, instal ulang server, melakukan pembersihan total ransomeware & malware tersembunyi, implementasi backup otomatis dengan fitur anti ransomeware dan melakukan monitoring sampai maintenance secara bekala
Hasil: Server kembali aman dan bersih dari ransomeware, data penting terselamatkan, backup otomatis berjalan setiap hari, resiko serangan menurun 90%, operasional perusahaan kembali normal dan infrastruktur keamanan meningkat.
